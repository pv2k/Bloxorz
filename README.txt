#Bloxorz
To complete a level:
you have to position the block in correct position on floor by moving or rotating it

To run the game:
1) Go to the GLFW directory.
2) run the following commands:
      i) make
      ii) ./Sample2D

ADDITIONAL LIBRARIES INCLUDED:-
	=>mpg123
	=>libao-dev

CUBOID BLOCK=> IT IS THE PRINCIPAL ELEMENT OF THE GAME.IT IS MADE OF TWO CUBE OF EQUAL DIMENSIONS

Extra Features:
1. background music will be played when block moves
2. no. of moves has been displayed.
3. TELEPORT SWITCH WHICH TELEPORTS THE CUBE INTO TWO POSITIONS COMPLETELY.
4. Level no. is specified
